Purrject Rebuild A Fortnite Lego Island

Project Description

In the story the players befriend autistic characters and learns about their strengths and challenges. Learning through interaction rather than instruction. Rebuilding the world requires honoring every way of thinking.

My Contributions

Game Programmer

Designed a cat prop that activates when the player is nearby and guides them to an NPC's accommodation, assisting the NPC.

Introduced a multi-array data structure using UEFN's UI Canvas API to add storytelling elements that contribute to world-building.

Through multiple playtests, fixed a bug related to the Xbox controller losing focus on UI buttons when navigating through multiple UIs. Instead of activating the UI only through Verse Script, utilized UEFN's Pop-up device to ensure players could interact with NPC Dialogue UI using either an Xbox controller or keyboard.

Created a resource manager using UEFN's prop manipulator device to trigger an event whenever the player damaged a Lego block. When this event was triggered, the number of stones received from the block was stored locally and made available for the player to use as currency to build essential props for further game exploration.
